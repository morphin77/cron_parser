================
Cron Parser for Python
================
----------
Description
----------

Parse CRON-like expression with Python
Parse and validate CRON expression according to https://en.wikipedia.org/wiki/Cron#CRON_expression

----------
Installation
----------
Go to project directory and execute
.. code-block::
    poetry install
    poetry run module
